# CNN Digit Classifier

A convolutional neural network (CNN) built with PyTorch that classifies handwritten digits (0–9) using the MNIST dataset.

## Overview

This project implements a deep learning pipeline for image classification. A CNN is trained on the MNIST dataset to recognise handwritten digits and predict labels for unseen images.

The model learns spatial features such as edges and shapes using convolutional layers, followed by fully connected layers for final classification.

## Key Features

* Convolutional neural network (CNN) architecture
* Trained on the MNIST dataset (handwritten digits 0–9)
* Uses PyTorch for model building and training
* Evaluates performance on a test dataset

## Tech Stack

* Python
* PyTorch
* torchvision
* Neural network modules (Conv2d, Linear, ReLU, MaxPool2d)

## Purpose

This project was built as a learning exercise to understand how neural networks are constructed and trained using PyTorch, and how image classification models work in practice.

This project was built by following a tutorial on YouTube by PyAI - Learn Python and AI.
